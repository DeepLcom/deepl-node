Hello Good Afternoon!

This is an example of how to translate a webpage using the DeepL Node.js library.

The Node.js script in this example translates the English version of this page into other languages, and the buttons below lead to those translated pages.

To run the Node.js script, run the following in the website-translation directory.
npm install
export DEEPL_AUTH_KEY=<your auth key>
npm run main

After the translated pages are generated, you can use the buttons below to view this page in your preferred language.

English German Spanish Bulgarian Italian
